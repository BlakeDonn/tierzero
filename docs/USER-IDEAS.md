# Tier Zero — User Feature Ideas (Raw)

Captured verbatim from user on 2026-02-26.

---

## Profession / Faction / Faction Filtering

In the bis list, or gearing tracker even, I want something where you can put like what professions you have, if you are aldor or scryer, if you are horde or alliance, and this will filter out different things.

## Auctionator Gold Pricing Integration

Also, I know auctionator has a 'full scan' feature that allows us to search for what current values are in gold, so we could essentially price out how much boes are, how much it costs to create tailoring items, how much it costs to increase rep to a point needed to buy an item, all kinda stuff like that, this will be very useful.

## Addon Rep Tracking

Also is it possible for our addon to check current rep standings? We should add that if possible.

## BiS List Accuracy & "Other Recommendations"

Also, wowhead sometimes has multiple 'bis' items for a category, I think we want to show which ones are deemed bis, also there is an 'other recommendations' option that is able to be opened up in wowhead to show other really strong options for that slot — we need to bring those in. Also what are we using to deem 'order' in our bis list? It doesn't seem to match wowhead.

## User-Editable BiS List

We also want the 'bis list' to be adjustable for users, like there should be an edit mode so they can make their own unique bis list by adjusting what items are in what rank, so they can have accurate comparisons.

## Gearing Routes — Filtered + Actionable Plans

For gearing routes, these would be filtered by some of the new ideas I mentioned above, we should think about that as well. In gearing routes I also want a like "actionable plan" to where you can go there, click create plan, (can adjust for a nightly plan, weekly, idk) and be like I have 3 hours to play what should I do, this then takes your gear, looks for the biggest tangible upgrades, then recommends like doing black morass or something like that, does that make sense?

## Arcane / Fire Damage Stat Tracking

There are pieces of gear (like spellfire gear) that has + arcane and + fire damage, this needs to be tracked in the stats view as well.

## Gems & Enchants Support

We need to add in support for gems and enchants from the prebis list.

## Gear Balancer / Optimizer

We need to add a button to like 'balance' gear which will take the stat prio from our prebis lists, recommend different gear you have in bank or inventory, or gems, or enchants, to get to like hit cap, then prio your most important stat or whatever (need to take into account arcane and fire damage again) so a user can find their best available gear from what they have. This will require storing like what gear they have in bank or inventory on addon upload.

## Claude's Suggestions (User-Approved)

- Stat weights / gear scoring (show a "score" per item based on spec weights)
- Compare two items side-by-side
- Shareable URLs (encode spec + tracked gear in URL params)
- Export/share gear set as image or text
- Phase filtering (P1 vs P2 items)
- Gem/enchant recommendations
